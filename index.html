Surgical Tools Segmentation

This project involves a machine learning model that provides a segmentation mask of surgical tools. The model is an ensemble of six U-net models, each segmenting individual tools. The masks are then combined to obtain the final segmentation mask.

Live view
PCOS Detection

This project aims to detect PCOS using ultrasonic images of the ovaries. The method produces a segmentation mask highlighting the cysts and uses machine learning models to classify these masks as infected or healthy. The project introduces a novel method for segmentation of cysts in the ovaries termed as "Salt segmentation".

Research Paper
P-Unet model

Introduces a novel fusion technique for U-net models, achieving a 20% higher IoU score than individual models. This model connects the decoder of a U-net model trained on original dermoscopic images to another U-net model trained on dermoscopic images overlapped with its Grad-CAM heatmap to obtain a more accurate segmentation mask.

Research Paper
ORB Feature Detection

Project uses ORB as feature extraction and demonstrates the use of two matchers : FLANN matcher and BF ( Brute force ) matcher. BF provides real time video processing as opposed to FLANN matcher.
